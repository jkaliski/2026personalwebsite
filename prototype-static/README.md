# Static Prototype Archive

This folder contains the original static prototype files that existed before the Next.js/Sanity port.

- `index.html` is the canonical prototype used for the port.
- `index-editorial.html` and `index-paper.html` are alternate visual experiments.
- `images/`, `uploads/`, and `screenshots/` are preserved for reference and source material.

The live application source now lives at the project root.
